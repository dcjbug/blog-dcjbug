== Description ==

This plugin lets you put your tweets in your wordpress site. You can add it as a widget in your sidebar, you can customize the way it looks and what twitter account you want to show on your feeds. You can also show as many twitter accounts as you like.

You can read the full explaination on <a href="http://webgeeksolution.com/156/wgs-twitter-feeds/">wgs twitter feeds</a>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wgs-twitter-feeds` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Drag and drop the wgs-twitter-feeds widget on the widget area
1. Fill up everything, the most important is the twitter username
